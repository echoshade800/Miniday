DATA MODELS
=============================================================================

Event Object:
  {
    id: string,
    title: string,
    targetDate: ISO string,
    categoryId: string,
    isPinned: boolean,
    repeatRule: 'none' | 'daily' | 'weekly' | 'monthly',
    remind: boolean,
    createdAt: number
  }

Category Object:
  {
    id: string,
    name: string,
    icon: emoji string
  }

Storage Keys:
  - MiniDays_events
  - MiniDays_categories
  - userData
  - MiniDaysinfo

HOW TO RUN
=============================================================================

1. Install:
   npm install

2. Start development server:
   npm run dev

3. Test on:
   - Web: Press 'w'
   - iOS: Press 'i' (macOS only)
   - Android: Press 'a'
   - Mobile: Scan QR with Expo Go app

4. Build for production:
   npm run build:web
